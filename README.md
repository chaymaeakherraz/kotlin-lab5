# kotlin-lab5
##  Description
Projet Kotlin pour le TP/Lab 5.
Ce TP introduit la récursion en Kotlin à travers l’exemple de la factorielle.
On compare une approche itérative et une approche récursive, puis on visualise la pile d’appels avec des traces.
##  Contenu
Tâche 1 : Implémenter la factorielle avec une boucle (factorialIterative)
Tâche 2 : Comprendre la logique récursive (cas de base + appel récursif)

Tâche 3 : Implémenter la factorielle avec une fonction récursive (factorialRecursive)
![kotlin-lab5](https://github.com/user-attachments/assets/fc28d683-625c-4f82-9068-f9b4ffa09763)
Tâche 4 : Suivre l’exécution pas à pas (pile d’appels)
Tâche 5 : Tracer la récursion avec des println (factorialTrace)
![kotlin-lab5](https://github.com/user-attachments/assets/788bd3c6-643d-48c4-940c-07f5c716e133)

##  Utilisation
Cloner le dépôt et ouvrir le projet Kotlin (ou un simple fichier main.kt).

Exécuter les fonctions suivantes pour tester :

factorialIterative(n)

factorialRecursive(n)

factorialTrace(n)
